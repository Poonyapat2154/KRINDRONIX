# KAT Drone Information System

Responsive front-end prototype for KRIN GROUP / KRIN Aviation and Technology.

Included modules:
- Dashboard with filters and calendar
- Product catalogue: UAV, GCS, Payloads, Data Links
- Project management with เลขที่สัญญา, value, organization, progress, priority, and status
- Customers
- Documents
- Maintenance
- Reports
- Settings
- Add-project form
- CSV export
- Mobile sidebar

Open `index.html` in Chrome or Edge.

This is a front-end prototype. Permanent login, database storage, file uploads, permissions, and real-time collaboration require a backend such as Flask + PostgreSQL.
